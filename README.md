# pdf_merger

A simple Command Line Interface (CLI) project built with Python to merge multiple PDF files into a single PDF using the PyPDF2 library.

## Features

- Merge multiple PDF files
- Simple and beginner-friendly CLI
- Fast and lightweight
- Built using Python

## Technologies Used

- Python
- PyPDF2
